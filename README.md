# Crack SQL Interview in 50 Qs – (Leetcode-top-50-SQL-Questions)

This repository contains solutions for the LeetCode SQL 50 question set.
It is intended as a quick, structured resource to prepare for SQL interviews and revise core concepts.

What’s included : 
- Clean, well-commented solutions to all SQL 50 problems from LeetCode
- Coverage of key topics: joins, aggregations, subqueries, CTEs, window functions, and filtering
- Consistent formatting to make it easy to review before interviews

Use this repo to
- Practice and revise SQL concepts in a focused way
- Understand different approaches to common interview-style SQL problems
- Build confidence for data engineering, data analyst, and backend interviews
